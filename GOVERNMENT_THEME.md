# Government-Style Minimal Theme Applied

Inspired by: https://visa.educationmalaysia.gov.my/legal/privacy-policy

## Design Philosophy
✓ Clean, flat, minimal design
✓ No shadows or 3D effects
✓ Simple borders (1px solid)
✓ Professional government colors
✓ Generous whitespace
✓ Accessibility-focused

## Color Palette

### Primary Colors (Government Blue)
- **Primary**: #1e3a8a (deep blue - trustworthy)
- **Primary Light**: #3b82f6 (hover states)
- **Background**: #f5f5f5 (neutral gray)
- **Sidebar**: #ffffff (pure white)

### Text Colors
- **Dark**: #1f2937 (headings)
- **Body**: #4b5563 (readable gray)
- **Secondary**: #64748b (muted text)

### Status Colors
- **Danger**: #dc2626 (red)
- **Warning**: #d97706 (amber)
- **Success**: #059669 (green)
- **Info**: #0284c7 (blue)
- **Border**: #d1d5db (light gray)

## Design Elements

### No Shadows
- All box-shadows removed
- Flat, 2D design
- Focus on borders instead

### Simple Borders
- All borders: 1px solid #d1d5db
- Border radius: 4px (minimal rounding)
- Table borders: 1px solid
- Form inputs: 1px solid border

### Spacing
- Main padding: 24px
- Card padding: 20px
- More generous whitespace
- Clear content separation

### Typography
- Font: System fonts (Arial, Segoe UI)
- Base: 13px
- Headers: 15-20px
- Font weight: 500-600 (not too bold)

### Buttons
- Flat design (no shadows)
- 4px border radius
- Solid colors
- Simple hover: lighter shade

### Forms
- 4px border radius
- 1px solid border
- Focus: blue ring (government style)
- Simple, clean look

### Tables
- Light gray header (#f9fafb)
- 1px borders throughout
- 12px cell padding
- Light row borders

### Navigation
- Clean sidebar (no shadows)
- Simple hover background
- 3px left border on active
- Generous padding (10px 20px)

### Cards
- White background
- 1px border
- 4px border radius
- No shadows
- Hover: slightly darker border

## Benefits

✓ **Professional**: Government/institutional appearance
✓ **Accessible**: High contrast, clear text
✓ **Clean**: No visual noise
✓ **Fast**: No shadow calculations
✓ **Trustworthy**: Serious, credible look
✓ **Readable**: Clear hierarchy, good spacing

## Implementation

All shadows removed:
- No box-shadow properties
- No drop shadows
- No elevation effects
- Flat throughout

Simple borders everywhere:
- 1px solid #d1d5db
- Consistent throughout
- Clean separation

Professional colors:
- Deep blue primary (#1e3a8a)
- Government-appropriate palette
- Not flashy or bright
- Serious and credible

Generous spacing:
- 20-24px padding
- Clear section separation
- Breathing room
- Easy to scan

## Result
A clean, professional, government-style dashboard with flat design, simple borders, and institutional credibility - similar to official government portals.
